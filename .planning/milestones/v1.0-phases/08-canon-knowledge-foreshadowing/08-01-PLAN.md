**Tool signatures and behavior (7 tools):**

1. `get_canon_facts(domain: str) -> list[CanonFact] | GateViolation`
   - SQL: `SELECT * FROM canon_facts WHERE domain = ? ORDER BY created_at`
   - Returns list (may be empty — empty list is valid, not NotFoundResponse)

2. `log_canon_fact(fact: str, domain: str = "general", source_chapter_id: int | None = None, source_event_id: int | None = None, parent_fact_id: int | None = None, certainty_level: str = "established", canon_status: str = "approved", notes: str | None = None) -> CanonFact | GateViolation`
   - Append-only INSERT (no ON CONFLICT)
   - Columns: domain, fact, source_chapter_id, source_event_id, parent_fact_id, certainty_level, canon_status, notes
   - Use cursor.lastrowid; SELECT back by id; return CanonFact

3. `log_decision(description: str, decision_type: str = "plot", rationale: str | None = None, alternatives: str | None = None, session_id: int | None = None, chapter_id: int | None = None) -> StoryDecision | GateViolation`
   - Append-only INSERT into decisions_log
   - Columns: decision_type, description, rationale, alternatives, session_id, chapter_id
   - Use cursor.lastrowid; SELECT back by id; return StoryDecision

4. `get_decisions(decision_type: str | None = None, chapter_id: int | None = None) -> list[StoryDecision] | GateViolation`
   - Build WHERE clause dynamically from optional params; ORDER BY created_at DESC
   - Returns list (may be empty)

5. `log_continuity_issue(description: str, severity: str = "minor", chapter_id: int | None = None, scene_id: int | None = None, canon_fact_id: int | None = None) -> ContinuityIssue | GateViolation`
   - Append-only INSERT into continuity_issues
   - Columns: severity, description, chapter_id, scene_id, canon_fact_id
   - Use cursor.lastrowid; SELECT back by id; return ContinuityIssue

6. `get_continuity_issues(severity: str | None = None, include_resolved: bool = False) -> list[ContinuityIssue] | GateViolation`
   - Build WHERE clause: if not include_resolved add `is_resolved = FALSE`; if severity add `severity = ?`
   - ORDER BY created_at ASC; returns list

7. `resolve_continuity_issue(issue_id: int, resolution_note: str) -> ContinuityIssue | NotFoundResponse | GateViolation`
   - UPDATE: `SET is_resolved = 1, resolution_note = ?, updated_at = datetime('now') WHERE id = ?`
   - commit(); SELECT back; if not rows return NotFoundResponse(not_found_message=f"Continuity issue {issue_id} not found"); else return ContinuityIssue

**Critical rules:**
- Every tool calls `check_gate(conn)` FIRST, before any other DB access
- No `print()` anywhere — use `logger` only
- `await conn.commit()` after every write
- Use `cursor.lastrowid` for INSERT row ID (aiosqlite pattern)
