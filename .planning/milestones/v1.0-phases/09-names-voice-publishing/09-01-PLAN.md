TOOLS to implement inside `register(mcp: FastMCP) -> None`:

1. **check_name(name: str) -> NameRegistryEntry | NotFoundResponse**
   - `SELECT * FROM name_registry WHERE LOWER(name) = LOWER(?)`
   - If row found: return `NameRegistryEntry(**dict(row))`
   - If no row: return `NotFoundResponse(not_found_message=f"Name '{name}' is not in the registry — safe to use")`
   - NO check_gate() call

2. **register_name(name: str, entity_type: str = "character", culture_id: int | None = None, linguistic_notes: str | None = None, introduced_chapter_id: int | None = None, notes: str | None = None) -> NameRegistryEntry | ValidationFailure**
   - Plain `INSERT INTO name_registry (name, entity_type, culture_id, linguistic_notes, introduced_chapter_id, notes) VALUES (?, ?, ?, ?, ?, ?)`
   - Wrap in `try/except aiosqlite.IntegrityError` — return `ValidationFailure(is_valid=False, errors=[f"Name '{name}' already exists in the registry. Use check_name to verify before registering."])`
   - On success: `cursor.lastrowid`, `await conn.commit()`, SELECT back by id, return `NameRegistryEntry(**dict(row))`
   - NO check_gate() call
   - Add `import aiosqlite` at top of file

3. **get_name_registry(entity_type: str | None = None, culture_id: int | None = None) -> list[NameRegistryEntry]**
   - Build conditional WHERE clause using the established pattern:
     ```python
     conditions: list[str] = []
     params: list[object] = []
     if entity_type is not None:
         conditions.append("entity_type = ?")
         params.append(entity_type)
     if culture_id is not None:
         conditions.append("culture_id = ?")
         params.append(culture_id)
     where_clause = f"WHERE {' AND '.join(conditions)}" if conditions else ""
     sql = f"SELECT * FROM name_registry {where_clause} ORDER BY name ASC"
     ```
   - Return `[NameRegistryEntry(**dict(row)) for row in rows]`
   - NO check_gate() call

4. **generate_name_suggestions(culture_id: int) -> NameSuggestionsResult**
   - Two separate SELECT queries (not a JOIN):
     - `SELECT * FROM name_registry WHERE culture_id = ? ORDER BY name ASC`
     - `SELECT naming_conventions FROM cultures WHERE id = ?`
   - `existing_names = [NameRegistryEntry(**dict(r)) for r in name_rows]`
   - `linguistic_context = culture_row["naming_conventions"] if culture_row else None`
   - Return `NameSuggestionsResult(existing_names=existing_names, linguistic_context=linguistic_context, culture_id=culture_id)`
   - If no names for culture → empty list (not an error)
   - If culture not found → linguistic_context=None (not an error)
   - NO check_gate() call
