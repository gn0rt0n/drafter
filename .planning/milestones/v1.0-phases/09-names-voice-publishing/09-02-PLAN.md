TOOLS to implement inside `register(mcp: FastMCP) -> None`:

1. **get_voice_profile(character_id: int) -> VoiceProfile | NotFoundResponse | GateViolation**
   - `gate = await check_gate(conn); if gate is not None: return gate`
   - `SELECT * FROM voice_profiles WHERE character_id = ?`
   - Return `VoiceProfile(**dict(row))` or `NotFoundResponse(not_found_message=f"No voice profile for character {character_id}")`

2. **upsert_voice_profile(character_id: int, profile_id: int | None = None, sentence_length: str | None = None, vocabulary_level: str | None = None, speech_patterns: str | None = None, verbal_tics: str | None = None, avoids: str | None = None, internal_voice_notes: str | None = None, dialogue_sample: str | None = None, notes: str | None = None, canon_status: str = "draft") -> VoiceProfile | GateViolation**
   - Gate check first
   - Two-branch upsert pattern:
     - `profile_id is None`: `INSERT INTO voice_profiles (character_id, sentence_length, vocabulary_level, speech_patterns, verbal_tics, avoids, internal_voice_notes, dialogue_sample, notes, canon_status) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?)` — use `cursor.lastrowid` as new_id
     - `profile_id is not None`: `INSERT INTO voice_profiles (id, character_id, ...) VALUES (?, ?, ...) ON CONFLICT(character_id) DO UPDATE SET sentence_length=excluded.sentence_length, vocabulary_level=excluded.vocabulary_level, speech_patterns=excluded.speech_patterns, verbal_tics=excluded.verbal_tics, avoids=excluded.avoids, internal_voice_notes=excluded.internal_voice_notes, dialogue_sample=excluded.dialogue_sample, notes=excluded.notes, canon_status=excluded.canon_status, updated_at=datetime('now')` — new_id = profile_id
   - `await conn.commit()`
   - SELECT back by id, return `VoiceProfile(**dict(row))`

3. **get_supernatural_voice_guidelines() -> list[SupernaturalVoiceGuideline] | GateViolation**
   - Gate check first
   - `SELECT * FROM supernatural_voice_guidelines ORDER BY element_name ASC`
   - Return `[SupernaturalVoiceGuideline(**dict(row)) for row in rows]`
   - Empty list is valid (not NotFoundResponse)

4. **log_voice_drift(character_id: int, description: str, drift_type: str = "vocabulary", severity: str = "minor", chapter_id: int | None = None, scene_id: int | None = None) -> VoiceDriftLog | GateViolation**
   - Gate check first
   - Append-only `INSERT INTO voice_drift_log (character_id, chapter_id, scene_id, drift_type, description, severity) VALUES (?, ?, ?, ?, ?, ?)`
   - `cursor.lastrowid`, `await conn.commit()`
   - SELECT back by id, return `VoiceDriftLog(**dict(row))`

5. **get_voice_drift_log(character_id: int) -> list[VoiceDriftLog] | GateViolation**
   - Gate check first
   - `SELECT * FROM voice_drift_log WHERE character_id = ? ORDER BY created_at DESC`
   - Return list (empty list is valid — no drift is valid state)
