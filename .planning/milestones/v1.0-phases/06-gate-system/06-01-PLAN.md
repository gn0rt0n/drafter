**Tool 1 — get_gate_status():**
- No parameters
- `async with get_connection() as conn:` SELECT * FROM architecture_gate WHERE id=1
- If no rows: return `NotFoundResponse(not_found_message="Gate record not found (id=1). Run 'novel db migrate' and 'novel db seed minimal'.")`
- Build `ArchitectureGate(**dict(gate_rows[0]))`
- SELECT COUNT(*) AS cnt FROM gate_checklist_items WHERE gate_id=1 AND is_passing=0
- Return dict: `{"gate_id": gate.id, "is_certified": gate.is_certified, "certified_at": gate.certified_at, "certified_by": gate.certified_by, "blocking_item_count": count}`

**Tool 2 — get_gate_checklist():**
- No parameters
- SELECT * FROM gate_checklist_items WHERE gate_id=1 ORDER BY category, item_key
- Return `[GateChecklistItem(**dict(r)) for r in rows]` — FastMCP serializes as N TextContent blocks

**Tool 3 — run_gate_audit():**
- No parameters
- Verify gate exists (architecture_gate WHERE id=1); return NotFoundResponse if missing
- `audited_at = datetime.now(timezone.utc).isoformat()`
- For each `(item_key, query)` in `GATE_QUERIES.items()`:
  - `result = await conn.execute_fetchall(query)`
  - `missing = result[0]["missing_count"] if result else 0`
  - `is_passing = 1 if missing == 0 else 0`
  - INSERT INTO gate_checklist_items with ON CONFLICT(gate_id, item_key) DO UPDATE, using values from GATE_ITEM_META[item_key]
- `await conn.commit()`
- Fetch updated rows: SELECT * FROM gate_checklist_items WHERE gate_id=1 ORDER BY category, item_key
- Build list of GateChecklistItem, compute passing/failing counts
- Return `GateAuditReport(gate_id=1, total_items=33, passing_count=..., failing_count=..., items=items_list, audited_at=audited_at)`

**Tool 4 — update_checklist_item(item_key: str, is_passing: bool, missing_count: int = 0, notes: str | None = None):**
- Verify item exists: SELECT * FROM gate_checklist_items WHERE gate_id=1 AND item_key=?
- If not found: `return NotFoundResponse(not_found_message=f"Checklist item '{item_key}' not found. Run run_gate_audit first.")`
- UPDATE gate_checklist_items SET is_passing=?, missing_count=?, notes=COALESCE(?, notes), last_checked_at=datetime('now') WHERE gate_id=1 AND item_key=?
- `await conn.commit()`
- Fetch and return updated `GateChecklistItem(**dict(rows[0]))`

**Tool 5 — certify_gate(certified_by: str | None = None):**
- Count failing: SELECT COUNT(*) AS cnt FROM gate_checklist_items WHERE gate_id=1 AND is_passing=0
- If failing > 0: `return ValidationFailure(is_valid=False, errors=[f"{failing} checklist item(s) not passing. Run run_gate_audit first, then resolve all gaps."])`
- UPDATE architecture_gate SET is_certified=1, certified_at=datetime('now'), certified_by=?, updated_at=datetime('now') WHERE id=1
  - Use `certified_by or "system"` for the parameter
- `await conn.commit()`
- Fetch and return `ArchitectureGate(**dict(rows[0]))`

**Critical constraints:**
- No `print()` anywhere — use `logger.debug()` or `logger.info()` only
- certify_gate does NOT re-run audit — it reads current item states and trusts them
- run_gate_audit does NOT certify — separate operations
- Use `ON CONFLICT(gate_id, item_key) DO UPDATE` for gate_checklist_items upserts (not INSERT OR REPLACE)
