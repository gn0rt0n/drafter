**`chapter` command (CLEX-01):**
```python
@app.command()
def chapter(
    n: int = typer.Argument(..., help="Chapter number to export"),
    output_dir: str = typer.Option("./chapters", "--output-dir", help="Output directory"),
) -> None:
```
- Ensure output dir exists: `Path(output_dir).mkdir(exist_ok=True, parents=True)`
- Query: `SELECT ch.*, c.name AS pov_character_name FROM chapters ch LEFT JOIN characters c ON c.id = ch.pov_character_id WHERE ch.chapter_number = ?`
- If no chapter found: `typer.echo(f"Chapter {n} not found.")` and return (exit 0)
- Query scenes: `SELECT * FROM scenes WHERE chapter_id = ? ORDER BY scene_number`
- Build markdown using the LOCKED format:
  ```
  # Chapter {n}: {title or 'Untitled'}

  **POV**: {pov_character_name or '(unknown)'}
  **Timeline**: Day {start_day or '?'} — Day {end_day or '?'}
  **Location**: {primary_location or '(unknown)'}
  ```
  If scenes exist, append each as:
  ```
  ## {scene['title'] or f'Scene {scene["scene_number"]}'}

  {scene['summary'] or '(no summary)'}

  ---
  ```
  If no scenes: append `\n*(No scenes recorded)*\n`
- Write file: `out_path = Path(output_dir) / f"chapter_{n:03d}.md"`, `out_path.write_text(content, encoding="utf-8")`
- Confirm: `typer.echo(f"Written: {out_path}")`

```python
@app.command()

```
- SQL: `SELECT nr.name, c.name AS character_name FROM name_registry nr LEFT JOIN characters c ON c.id = nr.character_id WHERE LOWER(nr.name) = LOWER(?)`
- If no rows: `typer.echo("No conflict.")`
- If rows: print each as `Conflict: {row['name']} (character: {row['character_name'] or 'unassigned'})`
