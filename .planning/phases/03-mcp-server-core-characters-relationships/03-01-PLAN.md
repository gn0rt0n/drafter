Tools to implement (7 total):

**get_character(character_id: int) -> Character | NotFoundResponse**
- `SELECT * FROM characters WHERE id = ?`
- Return `NotFoundResponse(not_found_message=f"Character {character_id} not found")` if row is None
- Return `Character(**dict(row))`

**list_characters() -> list[Character]**
- `SELECT * FROM characters ORDER BY name`
- Return `[Character(**dict(row)) for row in rows]`

**upsert_character(character_id: int | None, name: str, role: str = "supporting", faction_id: int | None = None, culture_id: int | None = None, home_era_id: int | None = None, age: int | None = None, physical_description: str | None = None, personality_core: str | None = None, backstory_summary: str | None = None, secret: str | None = None, motivation: str | None = None, fear: str | None = None, flaw: str | None = None, strength: str | None = None, arc_summary: str | None = None, voice_signature: str | None = None, notes: str | None = None, canon_status: str = "draft") -> Character | ValidationFailure**
- When character_id is None: `INSERT INTO characters (name, role, ..., updated_at) VALUES (?, ?, ..., datetime('now'))` — omit `id` so AUTOINCREMENT fires; fetch the created row by `conn.lastrowid`
- When character_id is not None: `INSERT INTO characters (id, name, role, ..., updated_at) VALUES (?, ?, ..., datetime('now')) ON CONFLICT(id) DO UPDATE SET name=excluded.name, role=excluded.role, ..., updated_at=datetime('now')`
- Catch `Exception` from the INSERT, return `ValidationFailure(is_valid=False, errors=[str(e)])` on failure
- `await conn.commit()` after every write
- Fetch and return the upserted `Character(**dict(row))`

**get_character_injuries(character_id: int, chapter_id: int | None = None) -> list[InjuryState] | NotFoundResponse**
- First verify character exists: `SELECT id FROM characters WHERE id = ?`
- If chapter_id provided: `SELECT * FROM injury_states WHERE character_id = ? AND chapter_id <= ? ORDER BY chapter_id DESC`
- Else: `SELECT * FROM injury_states WHERE character_id = ? ORDER BY chapter_id DESC`
- Return `[InjuryState(**dict(row)) for row in rows]`

**get_character_beliefs(character_id: int) -> list[CharacterBelief] | NotFoundResponse**
- Verify character exists
- `SELECT * FROM character_beliefs WHERE character_id = ? ORDER BY created_at DESC`
- Return `[CharacterBelief(**dict(row)) for row in rows]`

**log_character_knowledge(character_id: int, chapter_id: int, knowledge_type: str = "fact", content: str = "", source: str | None = None, is_secret: bool = False, notes: str | None = None) -> CharacterKnowledge | NotFoundResponse | ValidationFailure**
- Verify character exists; return NotFoundResponse if not
- `INSERT INTO character_knowledge (character_id, chapter_id, knowledge_type, content, source, is_secret, notes, created_at) VALUES (?, ?, ?, ?, ?, ?, ?, datetime('now'))`
- `await conn.commit()`
- Fetch the inserted row by `lastrowid` and return `CharacterKnowledge(**dict(row))`
- Catch Exception, return ValidationFailure on DB error

**get_character_location(character_id: int, chapter_id: int | None = None) -> list[CharacterLocation] | NotFoundResponse**
- Verify character exists
- If chapter_id: `SELECT * FROM character_locations WHERE character_id = ? AND chapter_id <= ? ORDER BY chapter_id DESC`
- Else: `SELECT * FROM character_locations WHERE character_id = ? ORDER BY chapter_id DESC`
- Return `[CharacterLocation(**dict(row)) for row in rows]`

- Always annotate return types explicitly on every tool function
- No module-level imports of `mcp` from server — always use the passed-in `mcp` parameter
- `await conn.commit()` after every INSERT/UPDATE
