TOOLS to implement inside `register(mcp: FastMCP) -> None`:

1. **get_publishing_assets(asset_type: str | None = None) -> list[PublishingAsset] | GateViolation**
   - Gate check first
   - Build conditional WHERE: if asset_type is not None → `WHERE asset_type = ?`
   - `ORDER BY created_at DESC`
   - Return `[PublishingAsset(**dict(row)) for row in rows]`

2. **upsert_publishing_asset(title: str, content: str, asset_id: int | None = None, asset_type: str = "query_letter", version: int = 1, status: str = "draft", notes: str | None = None) -> PublishingAsset | GateViolation**
   - Gate check first
   - Two-branch upsert on id (PK — no named UNIQUE beyond PK):
     - `asset_id is None`: `INSERT INTO publishing_assets (asset_type, title, content, version, status, notes) VALUES (?, ?, ?, ?, ?, ?)` — use `cursor.lastrowid` as new_id
     - `asset_id is not None`: `INSERT INTO publishing_assets (id, asset_type, title, content, version, status, notes) VALUES (?, ?, ?, ?, ?, ?, ?) ON CONFLICT(id) DO UPDATE SET asset_type=excluded.asset_type, title=excluded.title, content=excluded.content, version=excluded.version, status=excluded.status, notes=excluded.notes, updated_at=datetime('now')` — new_id = asset_id
   - `await conn.commit()`
   - SELECT back by id, return `PublishingAsset(**dict(row))`

3. **get_submissions(status: str | None = None) -> list[SubmissionEntry] | GateViolation**
   - Gate check first
   - Build conditional WHERE: if status is not None → `WHERE status = ?`
   - `ORDER BY submitted_at DESC`
   - Return `[SubmissionEntry(**dict(row)) for row in rows]`

4. **log_submission(agency_or_publisher: str, submitted_at: str, asset_id: int | None = None, status: str = "pending", notes: str | None = None) -> SubmissionEntry | GateViolation**
   - Gate check first
   - Append-only `INSERT INTO submission_tracker (asset_id, agency_or_publisher, submitted_at, status, notes) VALUES (?, ?, ?, ?, ?)`
   - `cursor.lastrowid`, `await conn.commit()`
   - SELECT back by id, return `SubmissionEntry(**dict(row))`

5. **update_submission(submission_id: int, status: str, response_at: str | None = None, response_notes: str | None = None, follow_up_due: str | None = None) -> SubmissionEntry | NotFoundResponse | GateViolation**
   - Gate check first
   - Partial UPDATE using dynamic SET clause:
     ```python
     set_parts = ["status=?", "updated_at=datetime('now')"]
     params: list[object] = [status]
     if response_at is not None:
         set_parts.append("response_at=?")
         params.append(response_at)
     if response_notes is not None:
         set_parts.append("response_notes=?")
         params.append(response_notes)
     if follow_up_due is not None:
         set_parts.append("follow_up_due=?")
         params.append(follow_up_due)
     params.append(submission_id)
     await conn.execute(
         f"UPDATE submission_tracker SET {', '.join(set_parts)} WHERE id = ?",
         params,
     )
     await conn.commit()
     ```
   - SELECT back: `SELECT * FROM submission_tracker WHERE id = ?`
   - `if row is None: return NotFoundResponse(not_found_message=f"Submission {submission_id} not found")`
   - Else: return `SubmissionEntry(**dict(row))`
